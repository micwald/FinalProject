Changing color of light based on sound sensitivity

Use Setting

The idea is control light color with sound, it can be used for example in a living room or at the entrance place

Instructions

 1. If the light is off and if sound level exceeds to predefined threshold, based on intensity, light will switch on with corresponding color

 2. Sound sensitivity is divided into three levels

    If sound intensity exceeds 560, light will switch on with Green color

    If sound intensity exceeds 570, light will switch on with Blue color

    If sound intensity exceeds 580, light will switch on with Red color

 3. If sound intensity exceeds 560 and light is already switched on, it will turn off

Send

Enter above one of the following values and send it

  • 590
  • 575
  • 565
  • 562

Light Status

Light is off
